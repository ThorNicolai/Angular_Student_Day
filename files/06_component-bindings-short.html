Components and bindings

In this section we will create a component with bindings. In the bonus exercises we will create more components based on this our new knowledge.

FlightCardComponent

  1. Create a new component in the folder flight-booking called flight-card by utilizing the angular cli: ng generate component flightBooking/flightCard --spec=false -is=true

  2. Open the file flight-card.component.ts and implement the following properties and methods:

    // flights-booking/flight-card.ts

@Component({
    selector: 'flight-card',
    templateUrl: './flight-card.component.html'
})
export class FlightCardComponent implements OnInit {

    constructor() { }

    @Input() item: Flight;
    @Input() selected: boolean;
    @Output() selectedChange = new EventEmitter<boolean>();

    ngOnInit() {
    }

    select() {
        this.selected = true;
        this.selectedChange.next(this.selected);
    }

    deselect() {
        this.selected = false;
        this.selectedChange.next(this.selected);
    }

}

    Notice that we choose the selector flight-card.

  3. Open the template flight-card.component.html. Change the content:

    <!-- flight-booking/flight-card/flight-card.component.html -->

<div
class="card"
[ngStyle]="{'background-color': (selected) ? 'rgb(204, 197, 185)' : 'white' }">

    <div class="header">
        <h2 class="title">{{item.from}} - {{item.to}}</h2>
    </div>
    <div class="content">
        <p>Flight-No.: #{{item.id}}</p>
        <p>Date: {{item.date | date:'dd.MM.yyyy HH:mm'}}</p>

        <p>
        <button
            class="btn btn-default"
            *ngIf="!selected"
            (click)="select()">Select</button>
        <button
            class="btn btn-default"
            *ngIf="selected"
            (click)="deselect()">Remove</button>

        </p>

    </div>
</div>

    Take a look at the data binding expressions in this file
