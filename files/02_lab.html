Prerequisites

  • NodeJS in current LTS-Version (http://nodejs.org)
  • Visual Studio Code (https://code.visualstudio.com/)
  • Angular CLI (npm i -g @angular/cli)

Recommented Plugins

The following two plugins make working with Angular in Visual Studio Code easier.

  • Angular Language Service Plugin (https://marketplace.visualstudio.com/items?itemName=Angular.ng-template)

  • Angular Files Plugin (https://marketplace.visualstudio.com/items?itemName=alexiv.vscode-angular2-files)

Getting started

In this section, we will take a closer took on the starter kit to get familiar with it.

  1. Extract the starter kit (starter.zip).

  2. Open the starter kit in Visual Studio Code.

  3. Look at the Component in the file app.component.ts and app.component.html to understand it’s functionality.

  4. Look at the Module in the file app.module.ts to understand it’s functionality.

  5. Look at the file index.html to understand its functionality.

  6. In case it has not already been done, start the development server:

    npm start

  7. Open the project in your browser.

  8. Change the property title in the file app.component.ts to Hello World:

    export class AppComponent {
    title = "Hello World!";
}

  9. Changing to the console, you can see that the application recompiles.

  10. Change to the browser to verify that the title is displayed as intended.

Component to search for flights

In this section we will implement the shown FlightSearchComponent.

  1. Create a folder entities in your project's app folder.

  2. In this folder, create a file called flight.ts:

    export interface Flight {
    id: number;
    from: string;
    to: string;
    date: string;
    delayed?: boolean;
}

  3. Create a folder flight-search in src/app for the FlightSearchComponent with the following files:

    • flight-search.component.ts
    • flight-search.component.html

    Hint: If you've installed the plugins mentioned above, you can right click the folder and select Generate Component.

    Hint: You can also generate those files using ng generate component flight-search

  4. Open the file flight-search.component.ts and assign the selector flight-search:

    import {Component} from '@angular/core';

@Component({
selector: 'flight-search',
templateUrl: './flight-search.component.html'
})
export class FlightSearchComponent {

}

  5. Open the file flight-search.component.html and insert some html to test it:

    <h1>Flight Search</h1>

  6. Make sure, the FlightSearchComponent is registered in the app.module.ts file.

    // app.module.ts

[...]
@NgModule({
declarations: [
    [...]
    FlightSearchComponent
],
[...]

  7. In app.component.html, replace the data binding to {{title}} with <flight-search></flight-search>

  8. Start your project (npm start). You should see the new component displayed.

Implement Component logic and connect it with the template

  1. Create following methods and properties for the FlightSearchComponent:

    
@Component({
    selector: 'flight-search',
    templateUrl: './flight-search.component.html'
})
export class FlightSearchComponent implements OnInit {

    from: string;
    to: string;
    flights: Flight[] = [];
    selectedFlight: Flight;

    constructor(private http: HttpClient) {
    }

    ngOnInit() { }

    search(): void {
        // Implementation follows
    }

    select(f: Flight): void {
        this.selectedFlight = f;
    }
}

  2. Implement the method search. It should use the properties from and to. Inject the HttpClient and use it to search for flights. Assign the found flights to the property flights.

    Show code 

    // flight-search/flight-search.component.ts

import {[...], HttpHeaders, HttpParams} from '@angular/common/http';

[...]

search(): void {
    const url = 'http://www.angular.at/api/flight';
    const params = new HttpParams()
                        .set('from', this.from || '')
                        .set('to', this.to || '');

    this.http
        .get<Flight[]>(url, {params})
        .subscribe(
        (flights: Flight[]) => {
            this.flights = flights;
        },
        (errResp) => {
            console.error('Error loading flights', errResp);
        });
}

  7. Add the data bindings to the inserted table.

    Show code 

    
    <div class="card">

        <table class="table table-contensed" *ngIf="flights.length > 0">
            <thead>
            <tr>
                <th>Id</th>

                <th>To</th>
